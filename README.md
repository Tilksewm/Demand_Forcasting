# Demand Forecasting ML Project

## Problem
Predict daily and monthly demand for supermarket products.

## Data
- Time-series sales data
- Multiple SKUs
- Holiday & seasonality effects

## Approach
- Feature engineering (lags, rolling stats, calendar features)
- Global ML model
- Evaluation using MAE & WAPE

## How to Run
```bash
pip install -r requirements.txt
python run_pipeline.py
